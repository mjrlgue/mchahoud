Internship: Data Analyst using Oracle tools

Before applying machine learning algorithm, we need data related to business to get insights from it, which is by building a data warehouse. In this internship, we were able to build one from scratch using an E-LT (ETL) based on ODI (Oracle Data Integration 11g) and an ORACLE stack on a Red Hat Enterprise Linux OS.

We started with selecting and defining our business approach, defining the project scope (using R. Kimball approach), then, modeling the start schema of our data warehouse, and defining the KPI's. Finally, for reporting and creating the dashboards for the end-user, we used OAT (Oracle Administration tool).
